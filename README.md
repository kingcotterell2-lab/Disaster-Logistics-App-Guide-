# RescueNet

RescueNet is a smart logistics and disaster response platform that connects citizens, volunteers, and organizations to coordinate deliveries, supplies, and emergency response in real time.

## Features

- Disaster reporting
- Live disaster map
- Volunteer coordination
- Supply logistics tracking
- AI damage analysis
- Automatic mission dispatch

## Tech Stack

Frontend:
Cursor AI
React
TailwindCSS
Mapbox

Backend:
Supabase
Node.js
Express

Database:
Supabase (PostgreSQL)

AI:
OpenAI API

## Running Locally

Frontend

cd frontend
npm install
npm run dev

Backend

cd backend
npm install
node server.js
